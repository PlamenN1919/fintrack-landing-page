# Фонови изображения

Тази директория е предназначена за съхранение на фонови изображения за FinTrack landing page.

## Поддържани формати
- PNG
- JPG/JPEG
- WebP
- SVG

## Препоръчителни размери
- Desktop: 1920x1080px или по-високо
- Mobile: 768x1024px или по-високо

## Именуване на файлове
Използвайте описателни имена като:
- `hero-background.jpg`
- `section-background.png`
- `mobile-hero-bg.jpg`

## Оптимизация
За най-добра производителност, оптимизирайте изображенията преди качване:
- Компресирайте файловете без загуба на качество
- Използвайте WebP формат когато е възможно
- Избягвайте файлове по-големи от 500KB
